# i-footer

A simple and responsive **Footer Component** built using **React (Create React App)**.  
This project is useful for learning component structure and can be reused in real-world web applications.


🔗 **Live Demo:** https://i-footer-h3rs.vercel.app/

<img src="https://drive.google.com/uc?export=download&id=12Ey91DMRqDbyJSyKAd5HX8ve4zCbBTHg">

This is a fully responsive footer component. Clicking on social media icons such as Instagram and Twitter redirects the user to their respective platforms.

---

## 🚀 Features

- Simple and clean footer design
- Fully responsive
- Built with reusable React components
- Beginner-friendly project structure

---

## 🛠 Tech Stack

- React.js
- Create React App
- JavaScript
- HTML5
- CSS3
